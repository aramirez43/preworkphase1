GIT is version software that we run on out computer.

GitHub is a remote application that works with GIT to help us collaborate and share our code with others.

sdc-web-student.cnm.edu is the bootcamp server that runs our code.

We add and commit our code to GIT on our computer.

We push our code from our computer to Github

We deploy our code from our computer to the bootcamp server.

deploy and push are different.
